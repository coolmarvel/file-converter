## P4 — 장기

- **macOS 빌드(arm64/x64 dmg)**: 현재 Windows(WSL) 환경이라 미실행. 맥에서 작업 시
  `docs/guides/packaging.md`(pdf-editor에서 이식) 절차대로 — ⚠️ 내부 번들명 ASCII(`FileConverter`) 필수,
  `scripts/dist-mac.cjs` 이식 필요, bgrm-data 354MB 포함 확인.

- ~~DICOM 미리보기~~ → **2026-07-13 무의미**: DICOM 기능 전체를 제거(전용 변환기 앱으로 분리 예정, 세션 8).
- ~~JPEG 품질 옵션 노출~~ → **2026-07-13 해결(v1.1.1)**: 컨텍스트 툴바 품질 슬라이더(jpeg/webp, 10~100%).
- **DICOM 전용 변환기 프로젝트**: 사용자가 별도 앱으로 특화 개발 예정 — 시작 시점·요구사항은 사용자 결정 대기.
  이 레포의 제거 직전 DICOM 구현은 git 이력(v1.1.0, `convert/dicom.ts`)에서 회수 가능.
- **GIF 애니메이션 보존**: 현재 GIF는 첫 프레임만 변환. 애니메이션 유지 변환은 인코더 라이브러리 필요.
- **인스톨러 다이어트(v1.3.0~)**: AI 모델 354MB 전체 번들 중 — small/medium 중 하나만 남기거나 선택 다운로드로 축소 검토 (사용자: "최적화는 나중에").
- **TIFF 다중 페이지**: 현재 첫 페이지만 디코드. 다중 페이지 TIFF → 여러 이미지/PDF 검토.
- ~~git 도입 검토~~ → **2026-07-09 도입 완료**(사용자가 직접 커밋·푸시). session-log SSOT는 유지 (세션 6, writing-guide).
