# 변경 이력

인스톨러 배포 단위의 사람용 요약. 최신이 위. 개발 진행 상세는 `session-log.md` 참고.

## [1.0.5] — 2026-07-08

- **PDF 편집(펜·텍스트 주석) 기능 제거** — 편집은 새 전용 앱 "PDF 편집기"(`~/pdf-editor`, v1.0.0~)가 담당.
  이 앱은 **변환 + 워터마크**에 집중.

## [1.0.4] — 2026-07-08 (사용자 확인 완료)

- **워터마크 켠 채 페이지 넘기면 꺼지던 문제 수정**: 워터마크 미리보기 캔버스가 확대 배율 × 윈도우 화면 배율(DPI)만큼
  무한정 커지던 것(400%+150% 배율이면 수백 MB)에 **해상도 상한**을 둠. 보이는 크기는 그대로.
- **페이지 넘김 방식 개선**: 넘기는 동안 이전 페이지를 계속 보여주고 작은 "불러오는 중…" 표시만 띄움
  (화면을 매번 부수고 다시 만들지 않아 더 안정적이고 덜 깜빡임).
- **자동 복구 안내**: 만에 하나 화면이 리셋되면 "자동 복구했습니다. 파일을 다시 추가해 주세요" 안내가 뜸
  (이전엔 조용히 리셋되어 앱이 꺼진 것처럼 보였음).

## [1.0.3] — 2026-07-08

- **대용량 PDF에서 페이지 넘기다 앱이 꺼지는 문제 수정** (30페이지 매뉴얼 등):
  - 페이지를 넘길 때마다 PDF 전체를 다시 읽던 구조를 → **문서를 한 번만 열고 페이지만 렌더**하도록 개편.
    넘김이 빨라지고, "다음"을 연타해도 메모리가 폭증하지 않음 (135MB·30페이지 테스트: 연타 시 1.4GB → 0.9GB 이내).
  - 스캔본처럼 페이지가 거대한 PDF도 미리보기 캔버스가 폭주하지 않게 해상도 상한 적용.
  - 만에 하나 화면 프로세스가 죽어도 **앱이 꺼지는 대신 자동으로 다시 로드**되는 안전망 추가(실제 강제 크래시 테스트로 확인).

## [1.0.2] — 2026-07-08 (설치 테스트 대기)

- **미리보기 확대 폭주 수정**: 확대하면 화면이 말도 안 되게 커지고 축소가 안 되던 문제의 진짜 원인
  (레이아웃이 내용 크기를 따라 무한히 넓어지는 되먹임)을 잡음. 이번엔 Playwright 자동 테스트로
  확대(140%·200%)/축소(40%)와 크기 수치를 실제 브라우저에서 검증함.
- **워터마크 미리보기 표시 정상화**: 같은 폭주 버그 때문에 오버레이 캔버스가 메모리 초과로 그려지지
  못하던 것 — 수정 후 대각선/바둑판 배치·확대 상태에서 실시간 표시되는 것 확인(스크린샷 검증).

## [1.0.1] — 2026-07-07

- 미리보기 **확대/축소** 정상화(안 되던 문제 수정), 편집 **텍스트 도구** 동작 개선.
- **워터마크**: 바둑판 배치 **간격 조절** 추가, **미리보기에 실시간으로** 보이도록.
- 크기(px) 가로×세로 입력 정렬 수정, 앱 상단 로고를 얼굴로.
- 설치 화면 헤더 글씨 겹침 수정, 앱/실행 파일 **아이콘 재생성**(더 호환되는 방식).
- **UI 리프레시**: 버튼·패널·입력 등 전반적으로 더 깔끔하게.

## [1.0.0] — 2026-07-07

- **앱 아이콘을 내 얼굴로**: 설치기뿐 아니라 실행 파일·앱 창·작업표시줄 아이콘까지 얼굴로 교체.
- **설치 화면 브랜딩**: 설치 마법사에 얼굴·서명·저작권(개인 저작물) 노출 + 라이선스 동의 페이지.
- **워터마크**: 변환 결과물에 텍스트/서명 워터마크(대각선·바둑판·모서리, 색·진하기·크기·기울기). DICOM 제외.
- **오른쪽 사이드바 스크롤**: 기능이 늘어도 옵션이 스크롤로 다 보이게.
- 버전 체계를 1.0.0으로 정식화(이후 수정마다 1.0.1, 1.0.2 …).

## [0.3.0] — 2026-07-07

- **미리보기 확대/축소 + 세로 스크롤**: 긴 PDF의 아래쪽 내용도 편집 가능(±·맞춤 버튼).
- **지우개**: 굵기만 조절해서 표시한 주석을 지움(원본 PDF는 불변).
- **제작자 브랜딩**: 앱 하단에 얼굴·서명·"제작 · 이성현 · © 2026".
- **앱 아이콘 교체**: 얼굴 배경 제거(누끼) 후 부드러운 원형 아이콘으로 설치기·앱 아이콘 반영.

## [0.2.0] — 2026-07-07

- **파일 순서 변경**: 목록에서 ▲▼로 변환 순서 조정(여러 장 → PDF 등에 반영).
- **PDF 편집 모드**: PDF/이미지 위에 펜(사인펜·네임펜·형광펜, 색·굵기·진하기)과 텍스트를 얹어 새 PDF로 저장. 페이지별로 독립 저장.
- 미리보기를 왼쪽 큰 영역(드롭존 아래)으로, 파일 목록을 오른쪽으로 이동.
- 이미지 변환 시 가로×세로 픽셀 리사이즈 옵션 추가(한쪽만 입력 시 비율 유지).
- 여러 이미지→PDF 및 PDF 파일에서 미리보기 페이지 넘김(‹ 1/N ›).

## [0.1.0] — 2026-07-07

- 최초 인스톨러 빌드. jpg → pdf 변환 사용자 테스트 성공.
- PDF↔이미지, 이미지↔이미지, 이미지→DICOM(SC) 변환 지원.
